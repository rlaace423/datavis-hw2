## Exploratory Data Analysis

### 주제: 가상자산 가격에 영향을 미치는 건 무엇이 있을까?

- Q1. 미국 달러 지수(DXY)는 가상자산 가격에 영향을 줄까?
- Q2. 화폐가 아닌 전통 자산(NASDAQ, GOLD)들도 가상자산 가격에 영향을 줄 수 있을까?
- Q3. 외부 요인이 아닌, 가상자산 내부 요인에 의한 변화가 존재할까?
